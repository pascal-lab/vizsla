Advanced is not a second user guide. It covers three kinds of work:

- Installation and build: source builds, local VSIX packages, prerelease packages, and custom language servers.
- Troubleshooting and logs: checking whether the language server started, finding command entry points, opening output channels, and generating profiling artifacts.
- Analysis model: understanding why an unconfigured workspace can still navigate code, and why diagnostics, rename, and Qihe change after `vide.toml` is configured.

For normal RTL editing, start with the [User Guide](../user-guide/). For feature usage, start with [Features](../user-guide/features/).
