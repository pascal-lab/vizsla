Advanced is not a second user guide. It covers two kinds of work:

- Installation and build: source builds, local VSIX packages, prerelease packages, and custom language servers.
- User configuration: language-server options for Neovim, Emacs, and other LSP editors.
- Troubleshooting and bug reports: filing issues and handling common startup, configuration, and refresh problems by symptom.
