Vide is a fully open-source modern SystemVerilog coding IDE developed by [PASCAL group](https://pascal-lab.net/) at Nanjing University, designed to make hardware design feel as fluid as software development. Vide brings more than a dozen code analysis capabilities that traditional hardware IDEs often lack, such as [Go to Definition](./features/navigation/), [Annotations](./features/annotations/), [Precise Completion](./features/completion/), and [Automatic Refactoring](./features/quick-fixes/). With Vide, hardware developers can understand, write, and maintain Verilog/SystemVerilog code more efficiently.

If you only want to get a feel for the core features, open the [Online Experience](./online-experience/) and try them without configuration. If you want to use Vide in your own project, start with [VS Code Installation](./vscode-installation/).
