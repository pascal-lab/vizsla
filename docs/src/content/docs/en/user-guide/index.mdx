Vide is a modern development environment for Verilog/SystemVerilog developers. It brings hardware projects closer to the editing experience software engineers already expect: go to definition, inlay hints, precise completion, automatic refactoring, diagnostics, and more. By helping the editor understand module hierarchy, port connections, macro definitions, and include relationships, Vide makes RTL code easier to read, write, and maintain.

If you only want to try the core features first, open the [Online Experience](./online-experience/) and use Vide in the browser without configuration. To use Vide in your own project, start with [VS Code Installation](./vscode-installation/).
