## What It Is

Signature help shows the target module's parameters or ports while the cursor is inside an instance parameter assignment list or port connection list. Completion inserts candidates; signature help tells you which parameter or port you are currently filling.

## When It Appears

- Inside parameter assignment lists `#(...)`, it shows target module parameters.
- Inside port connection lists `(...)`, it shows target module ports.
- Moving the cursor to another parameter or port updates the active item.

## Project Configuration

### Without Complete Configuration

If the target module is in the current file, signature help can be shown directly. If the target module is in another file, Vide needs to scan and resolve that module before it can show the corresponding ports and parameters.

### With Complete Configuration

After `vide.toml` is configured, signature help resolves target modules through `sources`, `include_dirs`, `defines`, and `libraries`. Parameter and port positions for cross-file instances become more accurate.

## Settings

To show only parameter-related signature help, enable:

```json
{
  "vide.signature.help.params.only": true
}
```

See [Signature Help settings](../../vscode-settings/#signature-help).
