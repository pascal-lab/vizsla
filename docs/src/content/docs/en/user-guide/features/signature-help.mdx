## What It Is

Signature help shows the target module's parameters or ports while the cursor is inside an instance parameter assignment list or port connection list. Completion inserts candidates; signature help tells you which parameter or port you are currently filling.

## When It Appears

- Inside parameter assignment lists `#(...)`, it shows target module parameters.
- Inside port connection lists `(...)`, it shows target module ports.
- Moving the cursor to another parameter or port updates the active item.

## Project Configuration

Signature help needs the target module to resolve. Like definition and completion, it depends on `sources`, `include_dirs`, `defines`, and `libraries` in `vide.toml`.

## Settings

To show only parameter-related signature help, enable:

```json
{
  "vide.signature.help.params.only": true
}
```

See [Signature Help settings](../../vscode-settings/#signature-help) for the full reference.
