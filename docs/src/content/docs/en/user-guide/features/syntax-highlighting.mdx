## What It Is

Syntax highlighting is lexical coloring for keywords, comments, numbers, strings, and basic syntax structures. It does not need full project analysis; it works as soon as you open a Verilog/SystemVerilog file.

Basic syntax highlighting, comment rules, bracket matching, and file icons are provided directly by the VS Code extension. After the language server starts, go to definition, references, hover, completion, diagnostics, semantic highlighting, inlay hints, outline, folding, signature help, and selection range become available.
