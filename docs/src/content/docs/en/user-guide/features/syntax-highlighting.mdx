## What It Is

Syntax highlighting is lexical coloring for keywords, comments, numbers, strings, and basic syntax structures. Basic syntax highlighting, comment rules, bracket matching, and file icons are provided directly by the VS Code extension; beyond grammar rules, it does not need additional analysis.
