## What It Is

Syntax highlighting colors keywords, comments, numbers, strings, and basic syntax structures. It does not need to understand the whole project first, so Verilog/SystemVerilog files get base coloring, comment recognition, bracket matching, and file icons as soon as they open.

## Project Configuration

None. Syntax highlighting does not depend on `vide.toml`. Once the Vide extension is enabled, Verilog/SystemVerilog files get basic syntax highlighting.
