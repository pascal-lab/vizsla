## What It Is

Selection range powers VS Code's `Expand Selection` and `Shrink Selection`. Plain text selection expands by characters; Vide expands by RTL syntax nodes, which is better for selecting expressions, port connections, parameter lists, instances, block statements, or whole modules.

## Project Configuration

Selection ranges mainly come from the current file's syntax tree, so they work in single-file cases. Project configuration makes macro branches and include-expanded structure closer to the real project, which improves the selection hierarchy.

## Related Features

Use selection range to edit the current code block. Use [Document Symbols](../document-symbols/) to browse file hierarchy, and [Folding](../folding/) to collapse structures.
