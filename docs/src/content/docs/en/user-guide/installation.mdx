Vide's core features are provided by its language server. The VS Code extension bundles and starts the language server automatically. Neovim and Emacs users need to download the Vide language server binary first, then configure the editor's LSP client to start it.

If you only want to try the core features first, open the [Online Experience](../online-experience/) without installing anything locally.
