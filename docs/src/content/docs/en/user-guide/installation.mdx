For daily use, install the stable Marketplace extension. After installation, open your RTL project directory; the extension automatically uses the bundled `vide` language server, so you do not need to configure a server path.

<ThinLinkCard
  href="https://marketplace.visualstudio.com/items?itemName=vizsla.vizsla-lsp"
  title="Visual Studio Marketplace"
  action="Open"
>
  Install stable Vide, extension ID: <code>vizsla.vizsla-lsp</code>
</ThinLinkCard>

You can also search for the display name `Vide` or the extension ID `vizsla.vizsla-lsp` in the VS Code Extensions view. After the extension is installed, continue with [Quick Start](../quick-start/).

## Nothing Happens After Installation

First check that the opened folder contains `.v`, `.vh`, `.sv`, `.svh`, or `.svi` files. The language features become visible when you open RTL files or an RTL workspace.

If the `Vide` status item does not appear, reload the VS Code window. If the status item shows an error or you need output logs, start with the [status bar section in Quick Start](../quick-start/#3-check-the-status-bar). Use [Advanced Installation](../../advanced-guide/advanced-installation/) and [Server Self-Check](../../advanced-guide/check-server/) when you need VSIX packages, a custom server, or startup logs.
