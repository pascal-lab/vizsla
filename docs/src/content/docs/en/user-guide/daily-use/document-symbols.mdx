## What It Is

Document symbols organize modules, ports, parameters, declarations, functions, tasks, generate blocks, and regions into a hierarchy. VS Code uses them in Outline, breadcrumbs, and `Go to Symbol in Editor`.

## Project Configuration

Document symbols mainly come from the current file, so they work when a single file is open. Project configuration affects include and macro-expanded structure and gives the file a more accurate project context.

## Related Features

Use document symbols to navigate structure. Use [Folding](../folding/) to collapse those structures, or [Selection Range](../selection-range/) to expand the current selection by syntax node.
