## Project Layout

Vide has two main parts:

- `vide`: the Rust LSP server. It reads projects, parses source files, maintains semantic information, and responds to editor requests.
- `editors/vscode`: the VS Code extension. It contributes Verilog/SystemVerilog language configuration, syntax highlighting, commands, a status bar item, and starts the language server.

Support for more editors is planned.

## Supported Files

The VS Code extension currently recognizes these file types:

| Language | Extensions |
| --- | --- |
| Verilog | `.v`, `.vh` |
| SystemVerilog | `.sv`, `.svh`, `.svi` |

The server also reads `.map` files when loading a project to support library maps and related project inputs.
