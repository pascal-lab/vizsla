The playground is a small editor that runs directly in the browser. You do not need to install the VS Code extension or prepare a local project. Open an example, edit the files, and try diagnostics, navigation, and completion in place.

It is useful for trying features quickly and reproducing small snippets. For your real project, go back to VS Code and configure `vizsla.toml`: this page cannot read files from your machine, run VS Code commands, run Qihe analysis, or use your local formatter.

To keep the documentation fast, the first example loads when it nears the screen. Later examples load only after you click them. The first load downloads editor assets, and later examples reuse the browser cache.
