Playground 会在当前页面中加载与独立 playground 相同的编辑器、示例工程和 WebAssembly 语言服务。为了避免拖慢普通文档页面，下面的示例默认按需加载：第一个示例接近视口后自动加载，其余示例需要点击后才会加载。

首次加载会下载 Monaco、Vizsla WASM 和 VS Code 语法资源；同一页面里的后续示例会复用浏览器缓存。普通文档页面只保留到本页的轻量链接，不会提前下载这些资源。

## 嵌入状态

文档页可以为每个示例设置初始状态，用来把读者直接带到要演示的 feature:

| 属性 | 作用 |
| --- | --- |
| `scenario` | 选择示例工程。 |
| `activeFile` | 打开指定 workspace 文件。 |
| `cursorLine` / `cursorColumn` | 把光标放到指定位置，使用 1-based 行列号。 |
| `selection` | 选中一段范围，格式为 `line:column-line:column`。 |
| `diagnosticsOpen` | 默认展开 diagnostics 面板。 |
| `load` | `click` 表示点击加载，`visible` 表示接近视口后加载。 |
