Playground 是一个可以直接在浏览器里运行的小型编辑器。你不需要安装 VS Code 扩展，也不需要准备本地工程；打开示例后，就可以在页面里改代码、看诊断、试跳转和补全。

它适合快速体验功能和复现小片段。真正接入你的项目时，仍然要回到 VS Code 中配置 `vizsla.toml`，因为本页不会读取你电脑里的文件，也不会运行 VS Code 命令、Qihe 分析或本地格式化工具。

为了避免拖慢文档，第一段示例会在接近屏幕时自动加载；后面的示例需要点击才会加载。第一次加载会下载编辑器资源，后面的示例会复用浏览器缓存。
