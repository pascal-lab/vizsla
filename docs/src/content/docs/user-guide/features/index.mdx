这里介绍 Vide 在 VS Code 中提供的所有功能，每个功能对应一个功能页。在每个功能页中，你会看到：

- 这个功能解决的问题；
- 一个示意图或者可以直接操作的演示；
- 如何通过 `vide.toml` 或 VS Code 设置调整行为。我们也在 [项目配置页](../project-configuration/#项目配置前后的行为差异) 中总结了项目配置前后的行为差异。

在阅读这一节前，请确保阅读过 [配置第一个项目](../first-project/)，因为有些深入分析的功能需要妥善编写项目配置才能正常工作。
