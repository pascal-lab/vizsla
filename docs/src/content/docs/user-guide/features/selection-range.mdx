## 什么是语义选区

语义选区接入 VS Code 的 `Expand Selection` 和 `Shrink Selection`。普通文本选择只按字符扩大；Vide 会按 RTL 语法节点扩大选区，更适合选中表达式、端口连接、参数列表、实例、块语句或整个模块。

## 项目配置影响

语义选区主要来自当前文件的语法树，单文件也能使用。项目配置会让宏分支和 include 后的结构更接近真实工程，从而让选区层级更准确。

## 相关功能

语义选区用于编辑当前代码块；如果要浏览文件层级，使用 [符号大纲](../document-symbols/)；如果要收起结构，使用 [折叠](../folding/)。
