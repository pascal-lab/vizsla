## 什么是语法高亮

语法高亮会给关键字、注释、数字、字符串和基本语法结构上色。在 Vide 扩展正常工作时，打开 Verilog/SystemVerilog 文件后就能立即看到基础着色、注释识别、括号匹配和文件图标。

## 项目配置影响

无。语法高亮本身不依赖 `vide.toml`。只要 Vide 扩展已启用，Verilog/SystemVerilog 文件就会应用基础语法高亮。
