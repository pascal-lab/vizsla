## 什么是折叠

折叠让编辑器按结构收起代码块。Vide 会为多行注释、region、module、端口列表、声明、generate/specify、实例、连续赋值、块语句和普通语句等范围提供 VS Code 代码行折叠。

在开发过程中使用折叠来跳过部分行的代码可以使代码界面更整洁。

## 项目配置影响

折叠主要来自当前文件的语法和结构解析，单文件也能使用。项目配置会影响 include、宏分支和语义解析后的文件结构，进而影响部分结构范围。

## 相关功能

如果你只是想快速跳到某个结构，使用 [符号大纲](../document-symbols/)；如果你想按语法结构扩大当前选择，使用 [语义选区](../selection-range/)。
