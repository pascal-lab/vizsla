## 什么是折叠

折叠让编辑器按结构收起代码块。Vide 会为多行注释、region、module、端口列表、声明、generate/specify、实例、连续赋值、块语句和普通语句等范围提供代码行折叠。

在开发过程中使用折叠来跳过部分行的代码可以使代码界面更整洁。

## 项目配置影响

### 没有完整配置时

折叠主要来自当前文件的语法结构，单文件也能使用。

### 有完整配置时

写好 `vide.toml` 后，include 文件和宏分支会按工程配置解析，相关结构范围也会随之更新。
