## 什么是代码注解

代码注解是 VS Code 的 Inlay Hints。Vide 会在有序端口连接旁显示目标端口名，在有序参数赋值旁显示目标参数名，并在 `endmodule` 等结构结尾处显示对应结构名。

这些提示不会改动源码；它们只是把目标模块和当前位置的关系直接显示在编辑器里。

## 项目配置影响

端口和参数注解需要目标模块能被解析到。没有项目配置时，Vide 使用当前文件和扫描结果；写好 `vide.toml` 后，提示会按真实工程范围解析模块、include 和宏分支。

## 调整显示

在 VS Code 设置里搜索 `Vide Inlay Hints`，或直接配置：

```json
{
  "vide.inlayHints.port.connection.enable": true,
  "vide.inlayHints.parameter.assignment.enable": true,
  "vide.inlayHints.end.structure.enable": true
}
```

完整参考见 [Inlay Hints 设置](../../vscode-settings/#inlay-hints)。
