Vide 是为 Verilog/SystemVerilog 开发者打造的现代 VS Code 开发环境，让编辑器真正理解硬件工程中的模块层级、端口连接、宏定义和 include 关系。Vide 把原本依赖搜索、经验和手动追线的日常工作，变成可以直接在代码里完成的跳转、补全、检查和重构；无论是接手一套陌生设计、补齐实例端口、追踪信号来源，还是结合 Qihe 分析真实工程问题，Vide 都把反馈收进编辑器的日常工作流里，帮助开发者更快读懂、修改和维护 RTL 代码。

如果你只是想先感受核心功能，可以直接打开[「在线体验」](./online-experience/)，无需配置即可体验核心功能；如果要在自己的工程里正式使用 Vide，请从[「VS Code 安装」](./vscode-installation/)开始。
