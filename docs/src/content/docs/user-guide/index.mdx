Vide 是由南京大学 [PASCAL 研究组](https://pascal-lab.net/) 研发的开源现代 SystemVerilog 编程 IDE，旨在让硬件设计像软件开发一样流畅。Vide 为硬件开发者带来了十多项传统硬件 IDE 中常常缺失的代码分析能力，包括[定义跳转](./features/navigation/)、[代码注解](./features/annotations/)、[精准补全](./features/completion/)和[自动重构](./features/quick-fixes/)等。借助 Vide，硬件开发者能够更加高效地理解、编写和维护 Verilog/SystemVerilog 代码。

## 从这里开始

如果你只是想先感受核心功能，可以直接打开[「在线体验」](./online-experience/)，无需配置即可体验核心功能；如果要在自己的工程里正式使用 Vide，我们推荐从[「VS Code 安装」](./vscode-installation/)开始，只需安装扩展就可以一键安装并启动 Vide 语言服务器，体验最完整、上手也最便捷；如果你使用 Neovim 或 Emacs，可以分别阅读[「Neovim 安装」](./neovim-installation/)和[「Emacs 安装」](./emacs-installation/)。

Vide 的核心是一个实现了 [Language Server Protocol (LSP)](https://microsoft.github.io/language-server-protocol/) 的语言服务器，因此理论上也可以适配更多支持 LSP 的编辑器。未来，我们会根据用户需求逐步扩展对更多编辑器的支持。
