进阶主要处理三类问题：

- 安装和构建：本地 VSIX、预发布包、自定义语言服务器和源码构建。
- 排障和日志：确认语言服务器是否启动、查看命令入口、打开输出通道和生成 profiling 产物。
- 分析模型：理解为什么无配置工程也能跳转，为什么写好 `vide.toml` 后诊断、重命名和 Qihe 行为会变化。

普通 RTL 编辑流程请从 [用户手册](../user-guide/) 开始；有关功能使用请看用户手册的 [功能特性](../user-guide/features/)。
